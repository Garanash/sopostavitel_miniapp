# Веб-приложение для сопоставления артикулов

React приложение для работы с системой сопоставления артикулов.

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## Переменные окружения

Создайте файл `.env`:

```
VITE_API_URL=http://localhost:8000
```

## Использование в Telegram

Для использования как мини-приложения в Telegram:

1. Соберите приложение: `npm run build`
2. Загрузите папку `dist/` на хостинг
3. Укажите URL в настройках бота через [@BotFather](https://t.me/BotFather)
