# Инструкция по использованию AI-поиска и подтверждения сопоставлений

## Настройка AI-поиска

Для работы AI-поиска необходимо добавить API ключ OpenAI в файл `.env` на сервере:

```bash
OPENAI_API_KEY=your_api_key_here
```

Модель по умолчанию: `gpt-3.5-turbo` (недорогая модель)

## Как работает AI-поиск

1. При обработке файла система сначала проверяет подтвержденные сопоставления
2. Если подтвержденного сопоставления нет, используется AI для интерпретации распознанного текста
3. AI ищет наиболее подходящую запись в базе данных, учитывая опечатки и вариации написания
4. Если AI не находит подходящей записи, используется обычный поиск по совпадению слов

## Подтверждение сопоставлений

1. После обработки файла в модальном окне результатов отображаются найденные сопоставления
2. Нажмите кнопку "✓ Подтвердить" напротив нужного сопоставления
3. Подтвержденное сопоставление сохраняется в базе данных
4. При следующем поиске того же текста система автоматически найдет его с 100% вероятностью

## Преимущества

- **Точность**: AI лучше интерпретирует распознанный текст, учитывая контекст
- **Обучение**: Система запоминает подтвержденные сопоставления
- **Экономия времени**: Подтвержденные сопоставления находятся мгновенно
